# スマホ・タブレットからのアクセスガイド

勤怠管理システムをスマートフォンやタブレットから簡単にアクセスできます。

## 📱 アクセス方法

### 方法1️⃣：直接URL入力（最も簡単）

#### 👤 従業員ポータル
```
https://kintai-fee67.firebaseapp.com/employee.html
```

#### 👨‍💼 管理画面
```
https://kintai-fee67.firebaseapp.com/kintai.html
```

ブラウザのアドレスバーにコピー＆ペーストして Enter キーを押してください。

---

### 方法2️⃣：ホーム画面からワンクリック（推奨）

スマホにショートカットを作成して、ホーム画面から直接アクセス可能にします。

#### iOS（iPhone/iPad）

1. Safari でアクセス：
   ```
   https://kintai-fee67.firebaseapp.com/employee.html
   ```

2. 下の共有ボタン（↑）をタップ

3. 「ホーム画面に追加」をタップ

4. ホーム画面にアイコンが追加されます

#### Android

1. Chrome でアクセス：
   ```
   https://kintai-fee67.firebaseapp.com/employee.html
   ```

2. 右上のメニュー（⋮）をタップ

3. 「アプリをインストール」または「ホーム画面に追加」をタップ

4. ホーム画面にアイコンが追加されます

## 📲 iOS/Android 別設定

### iOS の場合

**Safari の推奨設定：**
1. 設定 → Safari → アクセスのプライバシー → オン
2. これによりセッションが保持されます

### Android の場合

**Chrome の推奨設定：**
1. Chrome メニュー → 設定 → サイトの設定
2. JavaScript が有効になっていることを確認

## 🐛 トラブルシューティング

### ページが読み込めない

1. インターネット接続を確認
2. URL が正しいか確認
3. ブラウザキャッシュをクリア（設定 → プライバシー → キャッシュ削除）

### ログインできない

1. パスワードが正しいか確認
2. Caps Lock がオンになっていないか確認
3. ブラウザを再起動

### 打刻できない

1. インターネット接続を確認
2. ブラウザの JavaScript が有効か確認
3. 別のブラウザを試す

### データが同期されない

- インターネット接続が必要です
- 接続後、自動的に同期されます
- 手動更新：ブラウザを再読み込み（F5）
