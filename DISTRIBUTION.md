## 🆘 トラブル時の対応

### システムが利用できない場合

1. **インターネット接続確認**
   ```
   https://www.google.com にアクセス可能か確認
   ```

2. **ブラウザキャッシュクリア**
   - Ctrl + Shift + Delete
   - 「キャッシュされた画像とファイル」を削除

3. **別のブラウザで試す**
   - Chrome / Firefox / Safari など

4. **管理者に連絡**
   - IT 部門に報告してください

### ログインできない場合

- パスワードを確認
- Caps Lock が ON になっていないか確認
- ブラウザを再起動
