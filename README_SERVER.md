# 勤怠管理システム - ネットワークアクセスガイド

複数のPC・スマホから勤怠管理システムにアクセスするための手順です。

## 🚀 クイックスタート

### Windows

**この PC で server.bat をダブルクリック実行：**

```
server.bat
```

コマンドプロンプトが開き、IPアドレスが表示されます：

```
==================================================
    勤怠管理システム サーバー起動
==================================================

🚀 勤怠管理システムサーバー起動
==================================================

📍 ローカルアクセス:
   http://localhost:3000/

🌐 リモートアクセス:
   http://192.168.x.x:3000/

📋 管理画面:
   http://192.168.x.x:3000/kintai.html

👤 従業員画面:
   http://192.168.x.x:3000/employee.html

💡 スマホからアクセスする場合：
   http://192.168.x.x:3000/employee.html

🛑 終了: Ctrl+C を押してください
==================================================
```

### Mac / Linux

```bash
node server.js
```

## 📱 他の PC・スマホからアクセス

表示されたIPアドレスのURLをブラウザに入力します。

**管理画面（管理者用）：**
```
http://192.168.x.x:3000/kintai.html
```

**従業員画面（打刻用）：**
```
http://192.168.x.x:3000/employee.html
```

## 🔍 IPアドレスの確認方法

もし IPアドレスが見つからない場合：

### Windows
```
ipconfig
```
「IPv4 アドレス」を確認

### Mac/Linux
```
hostname -I
```
または
```
ifconfig
```

## ✨ 機能

✅ **リアルタイム更新**
- どのデバイスで操作しても、全デバイスに自動反映
- 複数ユーザーが同時にアクセス可能

✅ **複数デバイス対応**
- PC（Windows/Mac/Linux）
- スマートフォン（iOS/Android）
- タブレット

✅ **Firebase 同期**
- インターネット経由の同期にも対応
- オフライン時は localStorage で自動保存

## 🔧 ポート変更方法

3000番ポート以外を使いたい場合、`server.js` を編集：

```javascript
const PORT = 3000;  // ← この数字を変更
```

例：8080番ポートを使う場合
```javascript
const PORT = 8080;
```

## 🐛 トラブルシューティング

### URLに接続できない場合

1. **ファイアウォール確認**
   - Windows Defender ファイアウォールが Node.js をブロックしていないか確認
   - ブロックされている場合は許可を設定

2. **IPアドレス確認**
   - `ipconfig` で正しい IP アドレスを確認
   - `192.168.x.x` の形式が表示されているか

3. **ネットワーク接続確認**
   - この PC とアクセス元が同じ WiFi/ネットワークに接続しているか

4. **ポート確認**
   - ポート 3000 が別のアプリケーションで使用されていないか
   - 使用されている場合は `server.js` でポート番号を変更

### 画面が真っ白の場合

- ブラウザをリロード（F5 または Ctrl+R）
- ブラウザのコンソール（F12）でエラーを確認

### Firebase に接続できない場合

- インターネット接続を確認
- Firebase の設定が正しいか確認
- localhost でテストしてローカル動作を確認

## 📧 サポート

問題が発生した場合は、コンソールの出力メッセージを確認してください。
